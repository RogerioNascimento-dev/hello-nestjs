# Subdomínios

- Core: O que da dinheiro.
- Supporting: Da suporte para o core funcionar.
- Generic: Você precisa deles mas não são tão importantes.

## Exemplos

### Core

- Compra
- Catálogo
- Pagamento
- Faturamento
- Entrega

### Supporting
- Estoque

### Generic
- Notificação
- Promoções
- Chat
